About project

Наш сервис разработан для Trancparency International. Используя данные из declator.org, наш сервис строит визуализацию налоговых деклараций российских чиновников (мы используем данные из России, но их можно настроить для любого региона мира). Данные визуализации выбираются пользователем из списка групп структур правительства Также наш проект использует закон Бенфорда для поиска аномалий в данных. Это поможет найти коррупционеров
